# Contact Book System

## What it does
A complete contact book system built in C language.
Store and manage all your contacts permanently using file handling.

## Features
- Add new contacts with full details
- Display all contacts in formatted table
- Search contacts by name or phone number
- Update existing contact details
- Delete contacts
- Filter contacts by category
- Data saved permanently using file handling

## Categories
- Friend
- Family
- Work
- Other

## Concepts used
- Structures (struct)
- File handling (fread, fwrite, fseek, ftell)
- Functions and modular programming
- String operations (strstr, strcmp, strcspn)
- Menu driven program

## How to run
1. Compile:
gcc contact_book.c -o contact_book

2. Run:
./contact_book

3. Use menu to manage contacts

## Tech used
- C Language
- GCC Compiler
- File Handling
